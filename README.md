## Mvp - KANBAN

This project aims to simulate the KANBAN method digitally, organizing workflows using digital cards in 3 different stages: To Do; In Progress; Done. Each card contains three actions: move forward (Only in the first two stages); edit; and delete.

Este projeto busca simular o método KANBAN de forma digital, organizando fluxos de trabalho utilizando cartões digitais em 3 estágios diferentes: A fazer; Em Andamento; Feito.
Cada card contem três ações: avançar (Somente nas duas primeiras etapas); editar e excluir.
